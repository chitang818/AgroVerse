# 农资宇宙 (AgroVerse) 🌍

欢迎来到“农资宇宙”！这是一个将枯燥的农业知识、农资产品转化为生动形象的超级英雄宇宙的项目。
通过模块化的 AI 工作流，我们可以快速将农作物、病虫害、农药肥料等元素，转化为引人入胜的 15 秒短视频（如剧本、分镜、配音和 AI 绘画提示词）。

## 架构说明 ⚙️

这种架构就像开了一家电影制片厂，分为以下三大模块：

1. **[01_Characters (角色库)](./01_Characters/)** - **选角**：包含农资宇宙中各种角色的设定。
   - 🛡️ 守护者联盟 (Guardians)：各种肥料、农药（如：鱼蛋白、杀虫剂）。
   - 👾 破坏者联盟 (Destroyers)：各种病虫害（如：红蜘蛛、霜霉病）。
   - 🌱 大地子民 (Earthlings)：农作物、土壤环境（如：苹果树、干旱的土地）。
2. **[02_Workflows (工作流)](./02_Workflows/)** - **核心生产线**：
   - ✍️ `step1_script_writer` (编剧)：负责根据选定的角色，写出精彩的 15 秒、3幕剧的中文分镜脚本。
   - 🎬 `step2_prompt_maker` (导演/特效)：负责将中文分镜翻译成适用于 Midjourney/Runway 等工具的纯英文画面提示词（Prompt）。
3. **[Examples (案例展示)](./Examples/)** - **成片库**：
   - 生成的案例展示，如鱼蛋白抗冻的完整生成流程。

## 运作流程 (流水线) 🏭

1. **第一步（选角）**：从 `01_Characters` 中挑选出所需角色的设定（例如：“红蜘蛛”和“杀虫剂”）。
2. **第二步（编剧）**：将选好的角色设定丢给 `02_Workflows/step1_script_writer.md` 技能，它将输出极其精彩的 15 秒中文配音和画面描述。
3. **第三步（导演/特效）**：将上一步的脚本输入给 `02_Workflows/step2_prompt_maker.md` 技能，它作为纯粹的“画面翻译官”，把脚本转化为光影、镜头完美的英文 AI 视频/图像提示词。

这种模块化的设计，也是目前市面上最顶级的 AI 应用（如 Coze、Dify 中的工作流）的底层逻辑。
